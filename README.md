## üé® Customization

### Colors
The website uses BMW's signature colors defined in Tailwind config:
- **BMW Blue**: `#0066CC` (primary accent)
- **BMW Dark**: `#1a1a1a` (dark backgrounds)
- **BMW Gray**: `#2d2d2d` (card backgrounds)

### Animations
All animations are defined in `styles.css` and can be customized:
- **Duration**: Modify transition timings
- **Easing**: Change cubic-bezier functions
- **Delays**: Adjust stagger animations

### Content
Replace placeholder content in `index.html`:
- **Images**: Update car images with actual BMW photos
- **Text**: Customize model descriptions and company info
- **Videos**: Replace video src with actual BMW commercials
